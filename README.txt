== Description ==

This simple WordPress plugin allows you to effortlessly hide the NitroPack footer and remove the NitroPack footer badge from your website. Enhance your site's appearance without any hassle!

== Installation ==

Step 1: Download the Plugin from GitHub

    Go to the NitroPack Footer Remover GitHub repository (https://github.com/khmahfuzhasan/nitropack-footer-remover).
    Click on the Code button and select Download ZIP.
    Save the ZIP file to your computer.

Step 2: Install the Plugin

    Log in to your WordPress admin dashboard.
    Navigate to Plugins > Add New.
    Click on the Upload Plugin button.
    Select the downloaded ZIP file and click Install Now.

Step 3: Activate the Plugin

    Once the installation is complete, click the Activate button to enable the plugin.

Step 4: Clear NitroPack Cache

    Go to your NitroPack dashboard.
    Clear the entire cache for your website.

Step 5: Verify the Changes

    Open a private or incognito window in your browser.
    Visit your website to ensure that the NitroPack footer and badge are no longer visible.

If everything is set up correctly, the changes should be reflected!
